# Code Blocks

Create code blocks using triple backticks or indentation.

```python
function greet(name) {
  return `Hello, ${name}!`;
}
```

    def greet(name):
        return f"Hello, {name}!"

```java
public class HelloWorld {
    
    public static void main(String[] args) {
        System.out.println("Hello Java!");
    }
    
}
```
