# Getting Started

Drop Markdown files into `static/content` to add pages.

You can use plain files:

```txt
static/content/01.Getting Started.md
static/content/02.Guides/01.Installation.md
```

Or folder landing pages:

```txt
static/content/02.Guides/index.md
static/content/02.Guides/01.Installation.md
```
